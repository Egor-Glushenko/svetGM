# Настройка интеграции с Ozon API

## Получение API ключей

1. **Войдите в личный кабинет Ozon Seller**
   - Перейдите на https://seller.ozon.ru/
   - Войдите в свой аккаунт

2. **Перейдите в раздел API**
   - В меню выберите "Настройки" → "API"
   - Или перейдите по прямой ссылке: https://seller.ozon.ru/app/settings/api

3. **Создайте новое приложение**
   - Нажмите "Создать ключ"
   - Заполните данные приложения:
     - Название: "Свет GM - Сайт"
     - Описание: "Интеграция для автоматической загрузки товаров на сайт"
   - Выберите необходимые права доступа:
     - ✅ Чтение товаров
     - ✅ Чтение остатков
     - ✅ Чтение цен

4. **Получите ключи**
   - Скопируйте `Client ID`
   - Скопируйте `API Key`

## Настройка на сайте

1. **Откройте файл `js/ozon-api.js`**

2. **Замените пустые значения на ваши ключи:**
   ```javascript
   constructor() {
     this.apiUrl = 'https://api-seller.ozon.ru';
     this.clientId = 'ВАШ_CLIENT_ID'; // Замените на ваш Client ID
     this.apiKey = 'ВАШ_API_KEY'; // Замените на ваш API Key
     this.products = [];
   }
   ```

3. **Сохраните файл**

## Функциональность

После настройки API ваш сайт будет автоматически:

- ✅ Загружать товары с Ozon
- ✅ Отображать их в каталоге
- ✅ Показывать актуальные цены и остатки
- ✅ Обновлять информацию в реальном времени

## Ограничения Ozon API

- **Лимиты запросов**: 1000 запросов в минуту
- **Требуется авторизация**: Токен обновляется автоматически
- **CORS ограничения**: Может потребоваться прокси-сервер для продакшена

## Решение проблем

### Ошибка CORS
Если возникают ошибки CORS, создайте прокси-сервер или используйте серверную интеграцию.

### Товары не загружаются
1. Проверьте правильность API ключей
2. Убедитесь, что у приложения есть права на чтение товаров
3. Проверьте консоль браузера на наличие ошибок

### Медленная загрузка
- API может работать медленно при большом количестве товаров
- Рекомендуется кэширование данных

## Дополнительные возможности

### Автоматическое обновление
Можно настроить автоматическое обновление товаров каждые 30 минут:

```javascript
// Автоматическое обновление каждые 30 минут
setInterval(() => {
  ozonAPI.loadAndDisplayProducts();
}, 30 * 60 * 1000);
```

### Фильтрация товаров
Можно настроить загрузку только определенных категорий товаров, изменив параметры в методе `getProducts()`.

## Поддержка

При возникновении проблем:
1. Проверьте консоль браузера
2. Убедитесь в правильности API ключей
3. Проверьте права доступа в личном кабинете Ozon
